### BOT ANALYZER ###

- Objetivo do estudo
```
Determinar a assertividade do bot criado para dar sinais de jogos para aposta.
```

- Tecnologias utilizadas
```
- Python 3.9.0
- Pandas
- NumPy
```

- Como executar
```
- Criar um virtual environment
    python -m venv .venv
    source .venv/bin/activate
    make install
- Executar 
    python main.py import_files
```
